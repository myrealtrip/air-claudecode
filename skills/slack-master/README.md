# Slack Master

Slack 작업을 위해 slack-master 에이전트로 라우팅합니다.

## 사용법

```
/air-claudecode:slack-master <slack 작업>
```

## 기능
- 채널 메시지 및 스레드 읽기
- 공개/비공개 채널 메시지 검색
- 메시지 전송, 초안 작성, 예약 전송
- 채널 및 사용자 검색
- 사용자 프로필 읽기
- Slack 캔버스 생성 및 읽기
- Slack MCP 도구 (`mcp__slack__slack_*`) 사용
- 모든 쓰기 작업 (전송, 예약, 생성) 전 사용자 확인 필요
