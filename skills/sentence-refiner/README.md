# Sentence Refiner

[toss/technical-writing](https://github.com/toss/technical-writing)의 5가지 문장 규칙을 적용하여 한국어 기술 문서를 다듬습니다.

## 사용법

```
/air-claudecode:sentence-refiner .claudedocs/technical-writing/redis-guide.md
/air-claudecode:sentence-refiner 이 문서 문장 다듬어줘
```

## 기능

- 5가지 문장 규칙 적용: 주어 명확성, 간결성, 구체성, 자연스러운 한국어, 일관성
- 구조나 코드 블록을 변경하지 않고 기존 문서를 다듬기
- 변경 사항 요약 보고
